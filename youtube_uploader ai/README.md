# YouTube Автозагрузчик (v6) — без API и токенов

Загружает видео на YouTube через обычный браузер (Playwright).
Не нужны API-ключи, client_secret.json или токены — вход обычный, как в браузере.

## Установка (один раз)

```
pip install playwright
playwright install chromium
```

## Запуск

```
python youtube_uploader_gui.py
```

В заголовке окна должно быть написано "YouTube Auto Uploader v6".
Если написано другое — запустился старый файл, распакуйте архив заново.

## Как пользоваться

1. **Аккаунт** — "+ Добавить", войдите в открывшемся браузере любым способом и нажмите "Готово".
   Сессия сохраняется — повторно входить не нужно. Можно добавить много аккаунтов.
2. **Видео** — "Файлы" или "Папка". Для каждого можно задать заголовок, описание, теги.
   Кнопки "Описание ко всем" / "Теги ко всем" применяют текст сразу ко всем видео.
3. **Настройки** — доступ, категория. "Скрыть браузер" по умолчанию ВЫКЛ
   (YouTube часто блокирует скрытый режим — лучше оставить браузер видимым).
4. **Загрузить всё на YouTube** — автоматическая пакетная загрузка.


## 👥 Несколько аккаунтов одновременно

В списке аккаунтов выберите сразу несколько (Ctrl/Shift). При нажатии 'Загрузить всё' каждый аккаунт грузит свою копию очереди в своём окне браузера одновременно.

## 🖥 Ручная загрузка (гарантированный способ)

Если автоматика не срабатывает: выберите видео в очереди и нажмите "🖥 Ручная загрузка".
Откроется уже залогиненный браузер, файл подставится автоматически — вам останется дозаполнить и опубликовать вручную, потом закрыть окно.

## Диагностика ошибок

При сбое программа сохраняет скриншот в папку errors/ рядом с программой.
Откройте его — будет видно, на каком экране YouTube встала загрузка.

## Важно

Автоматизация зависит от вёрстки YouTube Studio. Если YouTube меняет интерфейс,
могут понадобиться правки селекторов. На этот случай есть "Ручная загрузка".


## 📊 Статистика (v8)

- Кнопка **“📊 Статистика”** рядом с кнопками загрузки.
- Выберите один или несколько аккаунтов и нажмите “Обновить из YouTube” — приложение откроет YouTube Studio в сохранённом входе и считает просмотры по каждому видео (без API).
- История загруженных через приложение видео сохраняется в `uploads.json`.

## Вход и активация (Этап 3)
При первом запуске программа просит войти по email и ввести ключ доступа:
1. Установите зависимости: `pip install -r requirements.txt` (добавлена библиотека `requests`).
2. Запустите `python youtube_uploader_gui.py` — откроется окно входа.
3. Вкладка «Регистрация» — создайте аккаунт (email + пароль). Затем введите ключ доступа.
4. После активации ключ привязывается к вашему аккаунту, и программа открывается.
5. Сессия сохраняется (`session.json`) — повторный вход не нужен, пока ключ активен.
6. Выйти из аккаунта можно во вкладке «Настройки» → «Выйти из аккаунта».

Без активного ключа программа не запускается (закрытый доступ).
